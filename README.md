# Enjoy sushita in command line! 


# About
・A cli tool.   
・You can play Typing game by run `$ sushita start`  
・Inspired by  ['sushida'](http://typingx0.net/sushida/), a typing game.  

# Run in local
```
**Get repository**  
$ git clone https://github.com/mox692/sushita  

**Quick start**  
$ go run .   
Let's run sushita start
```
